# dgdat to xlsx конвертер

Конвертирует базы ДубльГис [https://2gis.ru/](https://2gis.ru/) в файлы формата Microsoft Excel (csv).  
Кодировка - windows-1251, разделитель - точка с запятой.  
Все данные берутся только из файлов формата dgdat, обращение к интернету не требуется.  

## Установка

```
git clone https://github.com/mbry/DgdatToXlsx
composer update
```

## Сконвертировать все *.dgdat файлы в папке download

```
php convert.php
```

* Время работы
* Собственное название строения
